Platform Overview

X-AutoRaider is a sophisticated automation platform designed to manage multiple X (Twitter) accounts with intelligent quota distribution, AI-powered content generation, and advanced security features.

Smart Automation

Automated likes, retweets, and replies with intelligent rate limiting and error handling.

Quota Management

Dynamic quota distribution across accounts with real-time recalculation and optimization.

AI Integration

Security First

Intelligent Quota System

Global Pack Management

Premium Pack (1000 Actions)

Your premium pack includes 1000 total actions distributed intelligently across all connected accounts.

  • Total Actions: 1000 actions per pack
  • Daily Limit: Maximum 200 actions per day
  • Pack Type: Premium with advanced features
  • Expiry: No expiration date (lifetime pack)

Dynamic Distribution

Action Probabilities

  • Likes: 50% of daily quota (most common action)
  • Replies: 40% of daily quota (AI-generated responses)
  • Retweets: 10% of daily quota (selective sharing)

Automatic Recalculation

The system automatically recalculates quotas when:

  • New accounts are added via OAuth2
  • Accounts are removed or become inactive
  • Daily reset occurs (midnight UTC)
  • Manual quota adjustments are made
Important: When adding new OAuth2 accounts, automation automatically stops and displays: "Account X added, please restart automation" to ensure proper quota redistribution.

Troubleshooting

Common Issues

Frequently Asked Questions

Why isn't my automation working?

  • Check if automation is enabled in the dashboard
  • Verify accounts are connected and active
  • Ensure daily quota hasn't been exceeded
  • Check for error messages in the activity log

Account shows as "Muted" - what does this mean?

  • Account hit rate limits (429 errors) and is temporarily paused
  • System automatically unmutes after 15 minutes
  • Check activity log for specific error details
  • Account will resume normal operation automatically

Why did automation stop when I added a new account?

  • System automatically stops to recalculate quotas
  • This ensures proper quota distribution across all accounts
  • Simply restart automation to resume with updated settings
  • Message will display: "Account X added, please restart automation"

AI replies aren't generating - what's wrong?

  • Check AI token configuration in admin panel
  • Verify AI service is accessible and tokens are valid
  • Ensure reply probability is set above 0%
  • Check for AI-specific error messages in logs

Error Codes

Understanding System Messages

  • 403 Errors: Authorization issues - account paused for 15 minutes
  • 429 Errors: Rate limit exceeded - exponential backoff applied
  • Token Expired: OAuth token needs refresh - automatic retry
  • Quota Exceeded: Daily or total quota reached - wait for reset

Disconnecting AutoRaider from X (Twitter)

You can revoke AutoRaider's access to your X (Twitter) account at any time. Here's how:

  1. Open X (Twitter) and go to Settings & privacy.
  2. Click Security and account access, then Apps and sessions.
  3. Choose Connected apps and find AutoRaider.
  4. Click Revoke access to disconnect.
You can reconnect at any time by following your invitation link again.
